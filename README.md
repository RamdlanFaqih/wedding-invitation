## Installation

Install my-project with npm

```bash
git clone https://github.com/RamdlanFaqih/wedding-invitation.git
cd wedding-invitation
```

Install
```
npm install
```

Run dev server
```
npm run dev
```


Build (for production)
```
npm run build
npm run start
```
